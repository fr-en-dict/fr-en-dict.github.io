# French–English Dictionary

A free, ad-supported French–English dictionary web app powered by [Wiktionary](https://en.wiktionary.org).

## Features

- Definitions, etymology, conjugations, synonyms, derived terms, example sentences
- Content sourced live from the Wiktionary public API
- Installable as a PWA (Add to Home Screen)
- Responsive — works on desktop and mobile

## Data source

Content from [English Wiktionary](https://en.wiktionary.org) · [CC BY-SA 4.0](https://creativecommons.org/licenses/by-sa/4.0/)
